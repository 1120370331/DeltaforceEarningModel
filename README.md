# 基于三角洲收益指数的战术策略经济学评估研究

这是一个基于三角洲收益指数模型的交互式网页应用，用于评估不同战术策略的经济效率。
架构提示词
搭建一个简易的html页面来展示论文成果。第一步，用户第一次打开页面时，要将单位时间成本1分钟1万展示出来，并提供左右两个按键供用户调整这个自认为的单位时间收益基准（以及可以直接键盘输入数字修改）；第二步让用户设置情景，通过提供几个自认为身份选项来为默认模型参数提供一些变动，如同论文中提到的不同情境（选项：【跑刀鼠鼠】提高跑刀单局收益率增加30%，跑刀局撤离率+0.1，猛攻局撤离率降低0.1；【普通玩家】一切不变；【技术高手】猛攻局撤离率+0.1，收益+30%，对局时长-20%）。第三布，直接展示模型3指标各项玩法RANK，以及各项玩法基本数据。并且这里类似Excel允许访问者直接修改基础数值（在这里要体现上一步的增益指数，修改数值后依旧应用上一布选择的增益）。计算的排名区域和数据区域要分开（数据区域要通过点击编辑数据按钮来打开，在排名区域右上角的按钮，页面默认展示排名，并提供按照三个指标进行升降序，并按照简单聚类方法进行S/A/B/C/D五个分类（分位数聚类））
## 📊 项目简介

本项目实现了一个战术策略经济学评估系统，通过计算三角洲收益指数(RI*)来分析不同玩法的经济效率。系统支持：

- 时间机会成本参数(κ)调整
- 不同玩家身份的情景化设置
- 实时策略排名和数据分析
- 可编辑的基础参数

## 🎯 主要功能

### 1. 时间机会成本设置
- 调整时间机会成本参数κ（基准：1万哈夫币/分钟）
- 支持按钮调整和直接输入
- 实时更新所有计算结果

### 2. 玩家身份选择
- **跑刀鼠鼠**：跑刀收益+30%，撤离率+0.1，猛攻撤离率-0.1
- **普通玩家**：标准游戏体验，参数不变
- **技术高手**：猛攻撤离率+0.1，收益+30%，对局时长-20%

### 3. 策略排名分析
支持8种典型策略的经济效率评估：
- 全装猛攻、玻璃大炮猛攻、四套半装猛攻机密
- 绝密避战跑刀、经济学绝密跑刀
- 机密大坝速撤、普通经济学、全装单三猛攻

### 4. 核心指标计算
- **三角洲收益指数(RI*)**：成本-时间双重约束下的综合经济效率
- **每分钟期望净收益(J₁)**：强调"赚钱速度"，体现单局爆发能力
- **每单位成本期望净收益(J₂)**：强调"资本效率"，体现对钱包友好程度
- **风险调整后净收益(ΔR_adj)**：考虑成功率后的期望净收益
- **有效总成本(C_eff)**：显性成本与时间机会成本的总和

### 5. 交互功能
- 支持按各指标升降序排序
- 可编辑基础参数并实时计算
- S/A/B/C/D五级效率分类
- 响应式设计，适配不同设备

## 📐 模型原理

### 三角洲收益指数(RI*)计算公式

```
RI* = ΔR_adj / (C_eff × T)
    = (AR - AL) × SR / [(C + κT) × T]
```

其中：
- **AR**：单局平均营收额
- **AL**：单局平均损耗额
- **C**：显性入局成本
- **SR**：撤离成功率
- **T**：单局平均耗时
- **κ**：时间机会成本单价

### 分级标准
- **S级**：RI* ≥ 0.20（高效率）
- **A级**：0.05 ≤ RI* < 0.20（中高效率）
- **B级**：RI* < 0.05（低效率）

## 🔧 自定义配置

### 修改策略参数
点击"编辑数据"按钮，可以修改各策略的基础参数：

- 撤离率(SR)
- 耗时(T)

在JavaScript代码的`gameData.strategies`数组中添加新的策略配置。
